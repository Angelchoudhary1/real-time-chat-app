# 💬 Real Time Chat App

A full-stack real-time communication application that enables instant messaging between users.

## 🚀 Features
- Real-time messaging using Socket.io
- User authentication
- Responsive modern UI
- Live communication

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Socket.io

## 📌 Description
This project demonstrates real-time communication using WebSockets and modern full-stack technologies.
